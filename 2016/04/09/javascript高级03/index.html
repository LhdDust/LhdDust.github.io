2.Function

2.1 Function的概念说明

所有的函数都是Function的实例。

a.因为Object也是函数，可以推导出Object是Function的实例所以Object.__proto__指向Function.prototype。

b.因为Function也是函数，可以推导出Function也是Function的实例，所以Function.__proto__属性指向Function.prototype。

3.函数和对象的联系和区别

a.函数本质是一个对象，在浏览器端它和对象都具有__proto__属性，但函数具有propertype属性，对象没有

b.函数的__proto__属性最终指向Function.prototype，对象的__proto__属性最终指向Object.Property，因为函数的原型也是一个对象，它也指向Object.Property。

c.所有的函数都是Function的实例，Function本质上也是一个函数所以Function.__proto__等于Function.prototype

6.函数的静态属性

6.1 name

函数的name属性用于存储函数的名字，根据不同的函数形式，name的取值也不同，共有三种情况

a.声明式函数
声明式函数的name属性就是，name就是该函数名。

b.第二种函数表达式，并且函数表达式没有名字，name属性为变量名。

c.第三种函数表达式，并且函数表达式有名字，name属性为函数名，使用这种方式创建的函数只能使用变量名进行调用，想要使用函数名调用必须在函数内部。

6.2 形参的个数实参的个数

函数的形参个数，使用函数的length属性去获取。

函数的实参个数，在函数内部使用arguments.length属性去获取

6.3 caller

caller属性表示当前函数在哪个函数内部调用的。

若函数没有被调用，函数的caller的属性值为null;

若函数在全局作用域被调用，函数的caller的属性值为null;

若函数在其他函数内部被调用，函数的caller的属性值为该函数。

6.4 伪数组

对象具有length属性，并且有”0”,”1”,”2”这样的数字属性，可以使用for循环进行遍历，这样的对象可以被看作伪数组。

6.5 arguments.callee

在函数内部使用arguments.callee进行函数自调用
